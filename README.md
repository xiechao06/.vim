.vim
====

my vimrc

```
$ git clone https://github.com/xiechao06/.vim.git ~/.vim
$ cd ~/.vim
$ git submodule init
$ git submodule update
$ git clone https://github.com/Anthony25/gnome-terminal-colors-solarized
$ cd gnome-terminal-colors-solarized
$ ./install.sh
$ # after bundle installed
$ cd ~/.vim/bundle/tern_for_vim
$ sudo npm install tern
$ # install as https://github.com/suan/vim-instant-markdown instructs
$ # now edit ~/.vimrc.local if you want to override default settings
```
